## 0.0.1

- Initial Flutter port of [lochie/torph](https://github.com/lochie/torph).
- `DynamicTextMorph` widget with cubic-bezier and spring (`DynamicSpringParams`) easing.
- FLIP morph engine: per-grapheme/word gliding, enter/exit fade + scale,
  animated container resize, smooth mid-flight interruption.
- Reduced-motion / `disabled` support, optional debug outlines.
- Interactive playground in `example/`.
